Instrucciones para la ejecucion de aplicación
Se ejecuta desde ventana principal, donde se desplegara una interfaz que mostrara la lista de los restaurantes.
Se mostrara si el restuarante ha sido visitado y su califiación.

Se ofrecen dos opciones:
Nuevo Restaurante: Se abre una ventana, en donde el usuario puede poner la ubicación del restaurante,
el nombre del restaurante, la calificación (1-5), y si esta o no visitado.

Ver Restaurante: Se muestra una ventana, con todos los restaurantes, donde se separan por color, siendo
los verdes los que ya se han visitado y los rojos los que faltan por visitar, ademas de un filtro para mostar 
unicamente los visitados o no
